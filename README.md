# 🤖 AI Emotion Detector

A machine learning web application that predicts emotions from text using Natural Language Processing (NLP).

## Features

- 😊 Detects emotions from text
- 📊 Shows confidence score
- 🎨 Easy-to-use Streamlit interface
- ⚡ Built using Machine Learning

## Technologies

- Python
- Streamlit
- Scikit-learn
- TF-IDF
- Logistic Regression
- Hugging Face Emotion Dataset

## Example

Input:

I got a new laptop today and I am very excited.

Output:

😊 Joy

Confidence: 96%

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
